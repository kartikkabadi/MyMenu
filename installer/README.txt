MyMonitor installer
===================

1. Double-click “Install MyMonitor.command”.
2. The installer copies MyMonitor into /Applications (or ~/Applications if /Applications is not writable).
3. It removes the quarantine flag from this locally built, unsigned package and launches the app.
4. MyMonitor will show its own onboarding. Optional window tools ask for macOS privacy access only when enabled.

This package is ad hoc signed and is not notarized because notarization requires a paid Apple Developer account. For a source build, see the repository README.
